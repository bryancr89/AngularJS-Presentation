Services

Basically you should handle the app logic in services, then you can reuse the service through the controllers.

The services are Lazy instantiated & Singletons

You can create a service using the methods (service, factory & provider)

Example

			app.service( 'myService', function() {
  this.name = "default name";
});

app.factory( 'myFactory', function() {
  var name = "default name";

  return {
    name: name
  };
});

Directives

Are markers on a DOM (Attr, element name, comment or CSS class) that attach a specified behavior to that DOM element or even transform the DOM element and its children.

AngularJS provides directives like (ng-click, ng-repeat, etc).

Also you can create your custom directives.

Routing

Makes easy to wire together controllers, view templates, and the current URL location in the browser.

ngRoute & ui.router are the most popular modules

Example

			var app = angular.module('angularDemo',
  ['ngAnimate', 'ngCookies', 'ngTouch',
   'ngSanitize', 'ui.router']);

app.config(function ($stateProvider, $urlRouterProvider) {
  $stateProvider.state('home', {
    url: '/',
    templateUrl: 'partials/main.html',
    controller: 'MainCtrl'
  });
  $urlRouterProvider.otherwise('/');
});
